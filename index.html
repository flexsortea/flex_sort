flex_sort by flexsortea

Flexible sort, targeted for boost

This is the GIT repository for the boost::flex_sort, a work in progress to provide the Boost C++ Libraries with a flexible sort implementation

Install

header only To compile the test with VS, make sure the BOOST_PATH environment variable points to your boost directory. That's all.
